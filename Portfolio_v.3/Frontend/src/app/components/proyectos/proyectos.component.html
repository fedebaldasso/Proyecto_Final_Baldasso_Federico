Portfolio Personal

Uno de los proyectos realizados es mi Portfolio Personal creado en el curso de Full Stack Jr. brindado por Argentina Programa. Para llevar a cabo éste proyecto, en la parte de Front End se utilizó HTML, CSS, Bootstrap y Typescript a través del desarrollo en el framework Angular. En la parte de Back End se diseñó la base de datos en MySQL y se utilizó el lenguaje Java para el manejo de los mismos. El diseño de la API para conectar con el Front se realizó mediante Java Spring Boot. Finalmente el deploy del Front se realizó mediante Firebase y el del Back a través de Heroku conectando la base de datos con Clever Cloud.
